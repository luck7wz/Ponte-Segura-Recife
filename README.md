# 🚧 Ponte Segura Recife

Aplicativo desenvolvido para aumentar a segurança de pedestres na região do Recife.

## 📱 Funcionalidades

* Botão de emergência
* Envio de alerta com localização
* Integração com banco de dados
* Interface simples e rápida

## 🛠 Tecnologias

* HTML, CSS, JavaScript
* Node.js (backend)
* MySQL

## ▶️ Como rodar

### Frontend

Abra o arquivo `index.html`

### Backend

```bash
cd backend
node server.js
```

## 💾 Banco de dados

Execute o arquivo `database/banco.sql`
